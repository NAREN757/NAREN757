## 🎬 Featured Project

<div align="center">

### [FlimBox — Netflix-Inspired OTT Streaming Platform](https://flim-box-flax.vercel.app)

<img src="https://img.shields.io/badge/Live%20Demo-flim--box--flax.vercel.app-E50914?style=for-the-badge&logo=vercel&logoColor=white" />
<img src="https://img.shields.io/badge/Source%20Code-View%20Repo-141414?style=for-the-badge&logo=github&logoColor=white" />

</div>

A full-featured streaming web app built from scratch with **React + Vite**, powered by the **TMDB API**, with a UI engineered to look and feel like a real OTT platform.

| Feature | Detail |
|---|---|
| 🔍 **Search** | Real-time movie & show search across the TMDB catalogue |
| ▶️ **Trailer Playback** | Embedded trailer previews for any title |
| 🔐 **Authentication** | User sign-in flow for personalized access |
| 📌 **Watchlist** | Save and manage titles to watch later |
| 🎞️ **Genre Rows** | Netflix-style horizontal scroll rows by genre |
| 📱 **Responsive UI** | Fully responsive across desktop, tablet, and mobile |

**Engineering challenges solved:**
- Handled **TMDB API rate limiting** with efficient request strategies
- Implemented **dynamic routing** for movie/show detail pages
- Resolved **CORS** issues for clean cross-origin API calls
- Managed complex **UI state** (watchlist, auth, search) across components
- Configured **Vercel SPA routing** for clean deep-link navigation

```
React.js  +  Vite  +  TMDB API  +  React Router  +  Vercel
```
